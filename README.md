# Filterable Product Table – React Practice
A React exercise based on Thinking in React.
Implemented independently with some personal improvements.

# Features
List products grouped by category
Search products by name
Filter in-stock only
Highlight unavailable products
Added “In Stock” column (Yes / No)

# Key Concepts
Functional components & props
useState hook
Controlled inputs (input, checkbox)
Conditional rendering
filter() & map() for data processing

# Run Locally
npm install
npm run dev
